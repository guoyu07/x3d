Extensible 3D (X3D)

ISO/IEC 19775-1:2013

This document is Edition 3 of ISO/IEC 19775-1, Extensible 3D (X3D). The full title of this part of the International Standard is: Information technology — Computer graphics, image processing and environmental data representation — Extensible 3D (X3D) — Part 1: Architecture and base components.

Parts Description
Part 1: Architecture and base components Part 1 contains the abstract functional specification for the X3D framework, and definitions of the standardized components and profiles.
