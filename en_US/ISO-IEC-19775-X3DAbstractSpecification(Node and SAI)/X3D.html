Extensible 3D (X3D)

ISO/IEC 19775:2004

This document is ISO/IEC 19775:2004, Extensible 3D (X3D). The full title of the International Standard is: Information technology — Computer graphics and image processing — Extensible 3D (X3D).

Parts Description
Part 1: Architecture and base components Part 1 contains the abstract functional specification for the X3D framework, and definitions of the standardized components and profiles.
Part 2: Scene access interface (SAI) Part 2 contains the abstract functional specification for the Scene access interface (SAI) to the X3D runtime system.
