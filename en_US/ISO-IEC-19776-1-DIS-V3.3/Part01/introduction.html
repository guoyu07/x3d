Extensible 3D (X3D) encodings
Part 1: Extensible Markup Language (XML) encoding

Introduction

Purpose

Extensible 3D (X3D) is a system for describing interactive 3D objects and worlds as defined in ISO/IEC 19775. ISO/IEC 19776 specifies the X3D encodings of the functionality defined in ISO/IEC 19775. This part of ISO/IEC 19776 specifies the X3D encoding of X3D files using the Extensible Markup Language (XML).
